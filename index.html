About This Site

This website is built using pure HTML and CSS, showcasing modern web design techniques like CSS Grid, Flexbox, gradients, and animations. It's hosted for free on GitHub Pages and demonstrates how powerful static sites can be.

Technologies Used

HTML5 for structure, CSS3 for styling with modern features like backdrop-filter, CSS Grid, animations, and responsive design. No frameworks needed - just clean, semantic code that works everywhere.

Features

Responsive design that works on all devices, smooth animations, glassmorphism effects, modern gradient backgrounds, and optimized performance. Perfect for portfolios, landing pages, or documentation sites.
